Phone Book Guide

Bubble / Tag Legend

- Phone number
- Existing tag
- Saved tag
- Non existing tag

Dictionary

  • Tag - A word or phrase without a number or space that describes what you are looking for.
  • Bubble - The colored background surrounding and seperating the tags.
  • Autofill - A function of the Phone Book that automaticly predicts and completes any tag that you are typing.
  • Search Mode - A mode of Phone Book that allows you to search for a phone number by tags.
  • Edit Mode - A mode of Phone Book that allows you to edit numbers, create numbers, edit tags and create tags on a number.

Key Mappings / Shortcuts

Both Modes

  • Tab - Complete autofilled text.
  • Space - Create blank tag
  • End - Remove all tags and return to Search Mode
  • Delete - Remove selected tag.
  • Left Arrow - Select tag to the left. (If exists)
  • Right Arrow - Select tag to the right. (If exists)

Edit Mode

  • Up Arrow - Select tag input.
  • Down Arrow - Select description input.
  • Enter - Save changed data.

Searching Numbers

To search for a number just start typing.

While typing, you cannot type a tag that does not exist.

The autofill system will stop you from typing if the tag you are trying to type does not exist.

Editing Numbers

  1. Press [End] to clear your screen.
  2. Type in an existing Phone Number.
  3. Press [Space] to create a new tag
    • - A green tag indicates an existing tag on another number.
    • - A yellow tag indicates a saved/unedited tag already associated with this number.
    • - A gray tag indicates a non existing tag.
  4. Type a tag you would like to associate with this number.
  5. Use the [Down Arrow] key to select the "description" input.
  6. Type a description of the Phone Number.
  7. Press [Enter] to save your changes.
